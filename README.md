# Sign PDF FREE

A simple, lightweight browser-based tool to sign PDF documents. No server, no account — just open and sign.

All data is stored in browser memory and is lost on refresh, except for the signature saved in localStorage.

## Features

- **Draw signature** with color picker (black, red, blue, green)
- **Upload signature** image (PNG, JPG, SVG)
- **Add text** anywhere on the PDF
- **Drag & resize** placed elements
- **Save signatures** to localStorage for reuse
- **Download** the signed PDF

## Usage

1. Open `index.html` in your browser
2. Drop a PDF or click to browse
3. Add signatures or text — click on the page to place them
4. Drag to reposition, resize as needed
5. Click **Download** to save the signed PDF

## Tech

- [PDF.js](https://mozilla.github.io/pdf.js/) — PDF rendering
- [pdf-lib](https://pdf-lib.js.org/) — PDF modification
- No dependencies to install, no build step
